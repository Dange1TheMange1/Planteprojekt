Hello Gorgeous, this is Casper and Alba’s project idea that we think will blow you away by its sincere and amazing content. I will promise you this, this is like nothing you have ever seen before.
## General idea
The general idea of this project is a page that lists various fauna. When a plant is selected a 3D model of it will be displayed // a e s t h e t i c a l l y //, meaning that it will slowly spin and bob slightly. This is done through p5.js, which has the required capabilities to render 3D. Upon interaction, the plant will show its corresponding article on Wikipedia through  Wikimedia API.
## Stuff to use
JSON: To contain data about the plants.
P5.js javascript library: To render 3d models of plants.
Blender: To make the 3D models.
### Frameworks that are mandatory:
API’s: https://api.wikimedia.org/wiki/Getting_started_with_Wikimedia_APIs
Jquery: Because we are going to use Javascript, it might come in handy.
AJAX: It might come in handy.
## Scope
The site will feature 3 (there can be more) different plants, that all have the functionality described in the General idea section. There will be some popup windows for ads, and if you click on them a virus will “infect” the site.
